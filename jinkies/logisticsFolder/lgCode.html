There were three separate modes that you could run the lightning window in: autonomous, individual, and interactive. We did this so in our presentation we could show how our project would run without operator control, how each individual element is actuated, and how a user can control the major elements (thunder strikes, hand) whereas the minor elements (rain, wind) are taken care of autonomously.
